# Radixal
A Radix-notation based MIDI editor<br/>

## Controls
### Default Mode
`hjlk`: Left, Up, Down, Right
`0123456789ab`: Set active note (absolute) value
`+-v^`: Set active note (relative) value
`/`: Divide note/rest
`i`: Insert rest
`x`: Remove note
`.`: Unset note / Set 'Rest'
`:`: open command line
